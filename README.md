# Student Survey GUI Application

## 📌 Overview
This project is a Java Swing-based GUI application designed to manage student enrollment, course registration, and product creation using GoF design patterns.

## 🧠 Design Patterns Used
- **Singleton Pattern** – Enrollment system management
- **Factory Method Pattern** – Product creation (ProductA, ProductB)
- **Observer Pattern** – Course notifications to students

## 🖥️ Features
- Add and remove students
- Create and manage courses
- Enroll students in courses
- Notify students about course updates
- Create products using factory method
- GUI-based interaction using Java Swing
- Database integration using MySQL (JDBC)

## 🛠️ Tech Stack
- Java
- Swing (GUI)
- MySQL
- JDBC

## 🚀 How to Run
1. Clone the repository
2. Open in IntelliJ / NetBeans
3. Configure database connection in `DBConnection.java`
4. Run `MainGUIDB.java`
